# Webapp 

## Setup
* `npm install`

## Dev
* `npm start` 

And visit `http://localhost:8080`

# Depdendencies
* Dev Frontend:
  * webpack 
  * webpack-dev-server
  * ts-loader
  * html-webpack-plugin
  * clean-webpack-plugin
* Dev Backend: 
  * ts-node-dev
* Dev both: 
  * typescript
  * concurrently
* Exec frontend: 
  * react 
  * react-dom 
  * mobx 
  * mobx-react 
  * typestyle
  * csstips
* Exec backend: 
  * express 
  * lowdb
* Exec both: 
  * axios
